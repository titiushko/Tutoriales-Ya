55 - Pseudo-clases: only-child y only-of-type




Problema:

Desarrollar una página que muestre una tabla HTML con una fila. Fijar el color amarillo a la fila si tiene solo una fila la tabla. Modificar la tabla HTML agregando más filas y comprobar si ahora no se pinta la primer fila.

<!DOCTYPE html>
<html>
<head>
<title>Problema</title>
<link rel="StyleSheet" href="estilos.css" type="text/css">
</head>
<body>
<table>
  <tr>
    <td>fila 1</td><td>fila 1</td>
  </tr>
</table>
</body>
</html>

tr:only-child {
  background-color:red;
}

table {
  border-collapse: collapse;
  width:80%;
  margin:0 auto;
}

td {
  border: 1px solid #000;
  padding:0.5em;
}
